## Упражнение 16 01
## Тема: Импорт модулей в React
## Задача:
```JSX
Перед вами приложение, созданное с помощью Create React App. 
Мы удалили из него стандартный код. Теперь начнём создавать собственные компоненты.

Мы уже добавили папку components/ и поместили в неё два новых компонента Header и Main, но новый проект почему-то всё ещё не работает. 
Кажется, компоненты ничего не знают друг о друге. Давайте исправим это.

Также нужно перенести JSX-содержимое из App в Header, затем удалить значение App атрибута className. 
Заголовок стилизуем чуть позже.
После этого нужно импортировать Header и Main в основной компонент App и вставить новые компоненты в JSX: 
<Header /> и <Main />.

Не забудьте, что у JSX может быть только один внешний элемент, поэтому вам нужно вложить новые компоненты в родительский элемент.
```
